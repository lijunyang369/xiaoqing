# 小青回复模板

用于小青主进程的日常回复。具体回复逻辑以 `memory/long-term/BEHAVIOR-PROTOCOL.md` 第 4 章为准。

## 模板 1：任务执行中

```md
请求类型：<AI根据用户意图识别出的类型，对应 BEHAVIOR-PROTOCOL.md 3.2>
默认处理者：<与 3.2 对应的默认处理者>
当前任务：<TASK-ID> / <任务名> / <状态>

<本次任务响应正文>
```

## 模板 2：用户告知任务完成


# HEARTBEAT.md

# 配置心跳触发任务

## TOC 系统检查（主机制）

**当前状态**：已暂停（2026-04-09 09:39起）
**暂停原因**：TOC-001任务暂停，用户要求暂停相关心跳检查
**恢复条件**：TOC-001任务恢复时重新启用

### 设计原则
- **小青不阻塞**：所有耗时检查由 toc-monitor 子进程执行
- **快速响应**：小青在心跳时快速判断，决定是否触发子进程
- **异步处理**：子进程执行完成后通过 subagent_announce 通知小青

### 暂停期间规则
- **轻量检查**：跳过TOC队列状态检查
- **定时检查**：不触发任何TOC相关子进程（完整检查、日终复盘、周复盘）
- **异常检查**：即使发现队列异常，也不触发TOC监控
- **输出**：直接返回HEARTBEAT_OK，不执行TOC相关逻辑

### 检查流程

#### 1. 轻量检查（每 4 小时） - **小青执行**
**目标**：< 30 秒快速判断系统状态
**动作**：
- 快速扫描 TaskCard，统计各状态数量
- 检查 WIP 是否超限（Executing ≤ 3, WaitingDecision ≤ 2, Verifying ≤ 2）
- 如超限，立即触发相应暂停/清库存策略
- 如发现异常（队列异常增长、等待时间过长），触发完整检查

**输出**：
- HEARTBEAT_OK（如正常）
- 或触发 toc-monitor 子进程（如需要深入检查）

#### 2. 完整检查（每日 2 次：09:00, 16:00） - **触发子进程**
**触发条件**：到达预定时间或轻量检查发现异常
**动作**：
- 小青 spawn toc-monitor 子进程，任务类型为"完整检查"
- 子进程执行完整数据分析
- 子进程完成后通过 subagent_announce 通知小青

**子进程任务**：
```
检查类型：完整检查
重点关注：系统整体状态，制约识别，趋势分析
```

#### 3. 日终复盘（每日 20:00） - **触发子进程 + 小青主导**
**触发条件**：到达 20:00
**动作**：
1. 小青 spawn toc-monitor 子进程，任务类型为"日终复盘准备"
2. 子进程收集全天数据，准备复盘材料
3. 小青基于子进程输出执行复盘判断

**子进程输出**：
- 全天指标汇总
- 关键事件时间线
- 待回答的日终复盘五问数据

#### 4. 周复盘（每周五 17:00） - **触发子进程 + 小青主导**
**触发条件**：周五 17:00
**动作**：
1. 小青 spawn toc-monitor 子进程，任务类型为"周复盘准备"
2. 子进程收集本周所有数据，准备周复盘材料
3. 小青基于子进程输出执行周复盘五步法

**子进程输出**：
- 本周指标趋势
- 制约变化分析
- 实验效果评估
- 周复盘五步法输入数据

### 子进程触发规则
1. **轻量检查异常时触发**：队列超限、等待时间异常、返工率骤升
2. **定时触发**：09:00, 16:00 的完整检查；20:00 日终复盘；周五 17:00 周复盘
3. **手动触发**：俊阳或小青明确要求时

### 子进程管理
- **标签格式**：`toc-monitor - <检查类型> - <时间>`
- **任务模板**：使用 docs/spawn-task-templates.md 中的 toc-monitor 模板
- **结果处理**：子进程通过 subagent_announce 返回结果，小青更新系统状态
- **超时处理**：如子进程 10 分钟未完成，小青记录并考虑重新触发

### 小青快速检查脚本
为保持轻量检查高效，建议使用以下简化逻辑：
```python
# 伪代码
# TOC 检查暂停判断（2026-04-09 09:39起）
if TOC_系统暂停中:
    # 跳过所有TOC相关检查
    return HEARTBEAT_OK

# 正常TOC检查流程
if 当前时间 in [09:00, 16:00, 20:00] or (周五 and 17:00):
    spawn_toc_monitor(对应检查类型)
elif 队列超限() or 等待时间异常() or 返工率异常():
    spawn_toc_monitor("完整检查")
else:
    快速统计队列()
    检查WIP()
    return HEARTBEAT_OK
```

## 执行规则
- 如正在处理重要任务，可推迟非关键检查
- 夜间（23:00-08:00）只执行紧急检查
- 保持小青响应快速，复杂任务交子进程
- 子进程结果通过异步通知处理，不阻塞主交互

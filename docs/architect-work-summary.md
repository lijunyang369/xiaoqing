## 设计亮点

### 1. 分层配置系统
```
1. 内置默认值
2. 共享配置 (shared/openclaw.base.json)
3. 设备配置 (local/openclaw.local.json)
4. 环境变量 (.env)
5. 命令行参数
```

### 2. 智能目录分类
- **Shared**：必须同步（记忆、身份、任务）
- **Local**：设备特定（配置、日志、运行时）
- **Optional**：按需同步（大型技能、实验项目）

### 3. 完整的工具链
- 同步脚本支持 pull/push/both 三种模式
- 设备初始化脚本自动化设置
- 状态检查脚本监控同步健康度

### 4. 冲突安全处理
- 自动备份冲突文件（.conflict 后缀）
- 时间戳优先的自动合并
- 详细冲突日志记录

## 9. 任务相关回复模板

### 进行中

```md
当前任务：<TASK-ID> / <任务名> / In Progress
本次推进：<本条回复推进了什么>
```

### 已完成

```md
当前任务：<TASK-ID> / <任务名> / Done
本次推进：<完成了什么>
本进程其他进行中：<列表；无则写 无>
本进程其他未完成：<列表；无则写 无>
```

### 说明

- 普通聊天不强制使用这个模板
- 任务相关回复必须带“当前任务”
- 如果当前任务已完成，必须暴露其他未完成任务，防止多任务并行时失焦
